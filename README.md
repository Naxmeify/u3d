```
$ npm install -g u3d
```

## U3D
Unity3D CLI Tool
```
# Shows Help
$ u3d

# Init new Unity3D Project
$ u3d init

# Run Unity3D
$ u3d run

# Install Unity3D (current not available)
$ u3d install
```

## U3DAM
Unity3D Asset Manager CLI Tool
```
# Shows help
$ u3dam

# Init new AM Asset Project
$ u3dam init

# Install Dependencies
$ u3dam install

# Uninstall spcific assets
$ u3dam uninstall

# Update Dependencies
$ u3dam update

# Find Assets in AM
$ u3dam find <keywords>

# Setup login credentials for AM
$ u3dam login

# Publish new asset to AM
$ u3dam publish
```
